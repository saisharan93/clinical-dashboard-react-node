# Clinical Dashboard (React + Node)

A full-stack clinical monitoring dashboard built with React, Vite, and Node.js.

## Features
- Real-time backend health check
- Clinical patient table with vital signs
- Status-based visual prioritization
- Summary KPI cards for rapid triage
- Clean separation of frontend and backend

## Tech Stack
- Frontend: React, Vite, CSS
- Backend: Node.js, Express
- Dev Tools: Nodemon
